# Growth — Tech Innovation

A bold, modern variant with high-contrast colors for the Growth tier.

## Color Palette

- **Primary Accent**: `#006D79` (Venturo teal — used for brand elements only)
- **Electric Blue**: `#0066ff` — Headings, call-to-action elements
- **Neon Cyan**: `#00ffff` — Subtle accents, glow effects
- **Dark Gray**: `#1e1e1e` — Some dark elements for contrast
- **White**: `#ffffff` — Background
- **Subtle gradient overlay** using electric blue at 5% opacity

## Typography

- **Headers**: 'Boldonse' from canvas-fonts (looks tech-forward)
- **Body Text**: 'JetBrainsMono' or 'GeistMono' for technical feel

## Visual Treatment

- Subtle grid lines pattern in background (very low opacity)
- Electric blue under brand name as accent line
- Tech-mono font for module names and tags
- Cyan glow on hover elements (we're static so just on accent elements)
- Modern dashboard-mockup hint in meta-grid section
