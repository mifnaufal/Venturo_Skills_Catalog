# Enterprise — Tech Innovation

High-contrast bold tech aesthetic for cutting-edge enterprise solutions.

## Color Palette

- **Primary Accent**: `#006D79` (Venturo teal — always present)
- **Electric Blue**: `#0066ff` — Headings, brand emphasis
- **Neon Cyan**: `#00ffff` — Accent glow, badges
- **Dark Gray**: `#1e1e1e` — Main dark background
- **Pure Black**: `#0a0a0a` — Darker gradient tones
- **White**: `#ffffff` — Body text on dark

## Typography

- **Headers**: 'Boldonse' bold from canvas-fonts (tech-forward heavy font)
- **Body Text**: 'JetBrainsMono' for labels, 'Inter' for body
- **Code-like elements** for technical labels and tags

## Visual Treatment

- Pure dark gradient (black → dark gray)
- Electric blue and cyan gradient on brand name with strong glow
- Cyan outline checkmarks (matching neon modern)
- Tech-mono labels for team composition (e.g., "BA-001", "ENG-SR-001")
- Subtle scanning lines / grid pattern in background
- Box shadows with neon glow on key elements (budget box, brand name)
