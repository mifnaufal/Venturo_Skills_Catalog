# Starter — Modern Minimalist

A clean, contemporary variant of the Starter tier using grayscale + venturo teal accent.

## Color Palette

- **Primary Accent**: `#006D79` (Venturo teal — used for brand elements only)
- **Charcoal**: `#36454f` — Headings, key labels
- **Slate Gray**: `#708090` — Secondary text, dividers
- **Light Gray**: `#d3d3d3` — Background accents
- **Off White**: `#f9f9f9` — Main background
- **White**: `#ffffff` — Canvas white space

## Typography

- **Headers**: Use 'Outfit' bold from canvas-fonts
- **Body Text**: Use 'InstrumentSans' from canvas-fonts
- **Label text**: Tight tracking, uppercase, lighter weight

## Visual Treatment

- Very light gradient from off-white to pure white
- Single vertical accent line in venturo teal on left edge (2px)
- Subtle dot pattern or geometric grid as very faint background texture
- Checkmarks are outlined circles, not filled
- Feature list has generous spacing
