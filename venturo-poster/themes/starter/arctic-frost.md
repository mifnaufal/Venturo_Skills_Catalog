# Starter — Arctic Frost

A cool, crisp variant using ice blues and steel tones while keeping Venturo teal identity.

## Color Palette

- **Primary Accent**: `#006D79` (Venturo teal — used for brand elements only)
- **Ice Blue**: `#d4e4f7` — Light backgrounds
- **Steel Blue**: `#4a6fa5` — Headings, secondary accent
- **Silver**: `#c0c0c0` — Dividers, subtle accents
- **Crisp White**: `#fafafa` — Main background

## Typography

- **Headers**: Use 'Arriel SC' bold from canvas-fonts (or fallback to system sans-serif bold)
- **Body Text**: Use 'Inter' or system sans-serif

## Visual Treatment

- Ice blue gradient background (#fafafa → #d4e4f7 at bottom)
- Checkmarks have steel blue filled circles
- Subtle frost-like gradient overlay on brand name
- Very clean, airy spacing throughout
